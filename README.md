## Auctions App
[![Watch the video](https://img.youtube.com/vi/5zJ1cRa8OGs/0.jpg)](https://youtu.be/5zJ1cRa8OGs)

An app that allows users to register and then post auctions or bids on auctions made by other users. A SQL database is used to store data on users, listings, bids, comments, and each users' watchlist. Features user authentication and also uses Django forms to sync client side and server side validation with CSRF tokens for security. The companion Django Admin App has been configured to allow for admin oversight on user interactions. Tools/technologies used: Python, Django, SQL, HTML, Bootstrap, and CSS.

## GW2 Builds App
[![Watch the video](https://img.youtube.com/vi/UnuvSmzmWgk/0.jpg)](https://youtu.be/UnuvSmzmWgk)

This app started off as a spreadsheet of builds and quick notes and tips for builds that I liked to play or was interested in playing in Guild Wars 2 (abbreviate as GW2). I decided to turn it into a web app. It provides information on top builds for GW2 for WvW content. By utilizing the official GW2 API we have access to the most up to date in-game information. Users are able to leave comments on the builds to provide feedback and discussion. Tools/technologies used: Python, Django, SQL, Javascript, HTML, Bootstrap, and CSS.

## Social Network App
[![Watch the video](https://img.youtube.com/vi/fgUluRkogYg/0.jpg)](https://youtu.be/fgUluRkogYg)

An app that allows users to sign up to make posts and view posts from other users. By following other users, it is possible to create a custom feed instead of solely the main feed. Async Javascript is used to allow likes/dislikes to be done without a full page reload, and Pagination has been implemented to reduce the amount of server processing to not have to display the full amount of posts in the database. Tools/technologies used: Python, Django, SQL, Javascript, HTML, Bootstrap, and CSS.
